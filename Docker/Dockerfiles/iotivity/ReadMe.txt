#ReadMe
##After initial build / setup
###Android
Install Android SDK trough Android SDK Manager

###Path
Set the LD_LIBRARY_PATH in your .bashrc

##Build
scons TARGET_OS=android TARGET_ARCH=armeabi
Target Architectures: x86, armeabi, armeabi-v7a, armeabi-v7a-hard
